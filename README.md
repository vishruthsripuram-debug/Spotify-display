# Spotify Display
Physical spotify media player display with key-switch playback controls and a digital screen to view song info.

## BOM
Keyboard switches - Any hotswappable switches I used these ones ([https://www.aliexpress.com/item/1005006814061416.html](url))
MCU (microcontroller) - The microcontroller is the brains of the project. ([https://core-electronics.com.au/esp32-c3-mini-dev-board-iot-wifi-bluetooth.html](url))
TFT display - [https://www.aliexpress.com/item/1005009936908326.html](url) - The screen of the display.

## Printing instructions
- Print in the orientation of the display laying on the print bed
- Print with minimum 3 walls and 10% infill that is not lightning.
- Tolerances for the lid are tight as it is press fit it may be necessary to reduce speeds when printing to < 100 mm/s
